# 串口

Nuart提供两种通信模型。

模型1 ： Request-Reply，一问一答模式，发送请求-等待响应-收到响应 - 发送请求-等待响应-收到响应 - ...... - 发送请求-等待响应-收到响应.

上一次的请求收到响应或响应超时后，才会发送请求。

模型2：Fire-Forget，发送和接收响应在不同的2个线程中进行，互不干涉。
